## StringBuilder

> 문자열 출력에 사용

``` java
StringBuilder builder = new StringBuilder();

builder.append(strObject);
builder.append("\n");
	// Escape Sequence도 사용 가능
builder.toString(); 
	// 객체 타입을 StringBuilder에서 String로 바꿔주는 메소드
```

- Escape Sequence

  - `\r`(carriage return)

    - 스크린 커서의 위치를 현재 줄의 처음으로 옮긴다.

    - 이 기호다음에 오는 문자를 덮어씌운다. 

      - ``` java
        "첫 번째 시도\r두"	// 두 번째 시도
        ```

  - `\'` , `\"`
    - ' (작은 따옴표), " (큰 따옴표) 를 출력할 때 사용
